# 💒 Wedding Invitation Builder

A beautiful web-based tool to design digital wedding invitations inspired by the luxury green theme from Vietnamese wedding culture.

## ✨ Features

- **Section-based design** — Add, remove, and reorder invitation sections
- **Visual editor** — Edit content of each section with live preview
- **Image uploads** — Add photos to hero, love story timeline, and gallery sections
- **QR Code support** — Add QR codes in the RSVP section for guest confirmations
- **Background music** — Upload or link audio that plays when guests open your invitation
- **Drag & drop** — Reorder sections by dragging
- **Export** — Download your invitation as a standalone HTML file
- **Save/Load** — Save drafts to browser localStorage

## 📋 Section Types

| Section | Description |
|---------|-------------|
| **Hero / Cover** | Couple names, wedding date, optional cover photo |
| **Love Story** | Timeline of your relationship with photos |
| **Invitation Card** | Event details for groom's and bride's families |
| **RSVP** | Guest confirmation form with QR code area |
| **Thank You** | Closing gratitude message |
| **Photo Gallery** | Grid of your best photos |
| **Custom** | Free-form content block |

## 🎵 Background Music

The builder includes a music feature that:
- Supports MP3, OGG, and WAV files
- Auto-plays on first user interaction (browser requirement)
- Shows a floating toggle button for guests to mute/unmute
- Supports looping and external URL sources

## 🛠 Usage

1. Open the builder in your browser
2. Edit the default sections or add new ones
3. Upload your photos and QR code
4. Add background music (optional)
5. Click **Export HTML** to download your invitation
6. Share the exported HTML file or host it online
